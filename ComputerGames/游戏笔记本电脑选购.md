## 前期准备：明确需求

### 基础信息收集
在与大模型对话前，先收集以下信息：

**个人使用需求**
- 主要游戏类型（FPS、RPG、策略游戏等）
- 预期游戏画质设置（低、中、高、极高）
- 是否需要直播/录屏功能
- 除游戏外的其他用途（办公、设计、学习等）

**预算与偏好**
- 总预算范围
- 品牌偏好
- 尺寸偏好（13寸、15寸、17寸等）
- 重量要求（便携性 vs 性能）

## 分步骤需求分析指南

### 第一步：玩家类型确认与需求分析
**完整版提示词模板：**
```
我想购买游戏笔记本，请先帮我确认玩家类型，再分析配置需求：

=== 第一阶段：玩家类型确认 ===
我的游戏动机（可多选）：
- [ ] 竞争驱动：享受击败对手，追求排行榜名次
- [ ] 成就驱动：必须完成所有任务，追求100%完成度
- [ ] 探索驱动：喜欢发现隐藏内容，研究游戏机制
- [ ] 社交驱动：主要和朋友一起游戏，重视社交体验
- [ ] 经济驱动：关注游戏的投资回报和经济价值
- [ ] 技术驱动：喜欢尝试AI功能和最新技术

我的投入程度：
- 日均游戏时长：[具体小时数]
- 游戏经验：[新手/有经验/资深玩家]
- 硬件投入意愿：[3千以下/3千-8千/8千-1.5万/1.5万-3万/3万以上]

=== 第二阶段：具体游戏需求 ===
我主要玩的游戏：
1. [游戏名称1] - [画质要求：低/中/高/极高] - [帧率要求：60fps/120fps/144fps]
2. [游戏名称2] - [画质要求] - [帧率要求]
3. [游戏名称3] - [画质要求] - [帧率要求]

分辨率需求：[1080p/1440p/4K]
显示器刷新率：[60Hz/120Hz/144Hz/165Hz/240Hz]

请基于我的玩家类型，分析我的真实需求并推荐合理的配置等级，避免过度配置。
```

### 第二步：使用场景分析
**提示词模板：**
```
除了游戏，我还会用笔记本做以下工作：

- [ ] 图像处理（Photoshop、AI设计等）
- [ ] 编程开发（语言：[具体语言]）
- [ ] 3D建模渲染（软件：[具体软件]）
- [ ] 直播推流
- [ ] 学习网课
- [ ] 其他：[具体说明]

使用时长：每天[X]小时
主要使用场景：[家用/办公室/学校/移动办公]

基于这些需求，请推荐合适的配置，特别是内存、存储和CPU的要求。
```

### 第三步：预算合理性检查与便携性分析
**理性消费引导提示词模板：**
```
我的预算和便携性要求：

预算范围：[X]元 - [Y]元
可接受的重量：最大[X]kg
屏幕尺寸偏好：[13-14寸便携/15-16寸平衡/17寸大屏]
续航要求：至少[X]小时（非游戏状态）

品牌偏好：
- 优先考虑：[品牌列表]
- 不考虑：[品牌列表，如果有的话]

=== 重要：预算合理性检查 ===
请基于以下原则帮我分析预算是否合理：

1. **性能溢出检查**：我的配置需求是否超出实际游戏需要？
2. **成本效益分析**：高配置的额外投入是否有实际价值？
3. **替代方案建议**：是否有更经济的方案达到同样效果？
4. **总拥有成本**：4年使用期内的总成本（包括电费、配件、维护）
5. **贬值风险评估**：不同价位产品的保值能力对比

请特别提醒我避免以下误区：
- "顶级配置=更好体验"的迷信
- 忽视实际使用场景的硬件崇拜
- 被营销话术误导的冲动消费

同时请结合2025年游戏发展趋势，给出客观的配置建议。
```

```
请综合分析我的需求并给出配置建议：

=== 游戏需求 ===

=== 工作需求 ===
工作用途：[从第二步获得的使用场景]
配置要求：[从第二步获得的内存/存储要求]

=== 预算与理性分析 ===
预算：[具体金额]
便携性：[具体要求]
预算合理性：[从第三步获得的合理性分析]

请提供：
1. **基于玩家类型的精准配置**（避免过度配置）
2. **性价比最优方案**（结合2025年游戏趋势）
3. **三档选择**：
   - 满足需求的最低配置（经济型）
   - 推荐配置（平衡型）
   - 未来保值配置（投资型）
4. **理性消费提醒**：
   - 避免哪些不必要的高配置
   - 重点投入哪些核心配置
   - 可以通过软件/设置优化替代的硬件需求
5. **总拥有成本分析**：4年使用期的真实花费对比
```

## 常见问题解答

### Q: 16GB内存够用吗？
A: 对于纯游戏用户，16GB DDR5足够。如果需要视频剪辑、3D渲染或大量多任务处理，建议32GB。高端配置（RTX 4080/4090）建议64GB起步。

### Q: RTX 4060和RTX 4070差距大吗？
A: 性能差距约15-20%，4070在1440p游戏和创作任务中优势明显。预算充足建议直接4070。

### Q: RTX 4070Ti值得买吗？
A: 相比4070性能提升30%，显存增加到12GB，适合4K游戏入门和专业创作。如果预算在18000+，4070Ti是很好的选择。

### Q: RTX 4080和4090差距有多大？
A: 4090比4080性能高25-30%，显存16GB vs 12GB。4090适合追求极致性能和专业创作，4080已能很好胜任4K游戏。

### Q: AMD和Intel处理器怎么选？
A: AMD 7840HS功耗更低续航更好，Intel 13代多核性能更强。游戏性能基本相当。高端配置建议Intel i9-13980HX。

### Q: 要不要等RTX 5060？ ⏰ *预测基于2024年12月信息*
A: RTX 5060预计2025年Q2发布，性能提升预期10-15%。急用建议现在买，不急可以等。

### Q: 4K游戏需要什么配置？
A: 4K高画质60fps建议RTX 4080起步，4K极高画质建议RTX 4090。CPU推荐i9-13980HX，内存32GB+。

### Q: 高端游戏本散热够用吗？
A: RTX 4080/4090功耗很高，必须选择散热设计优秀的型号。建议选择双风扇三热管以上配置，避免公版散热方案。
